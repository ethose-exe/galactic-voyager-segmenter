### *AI-Powered Intelligence for the Future of Space Travel*

## 🎯 Project Overview
This project uses **Unsupervised Machine Learning (K-Means)** to segment interstellar travelers. By analyzing travel distance and luxury spend, we identify three key personas:
1. **Lunar Tourists:** Low spend, frequent short trips.
2. **Deep-Space Whales:** Ultra-high spend, long-distance pioneers.
3. **Tech Explorers:** High tech-upgrades, mid-range travel.
